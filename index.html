Using eZ Platform in an API era

The time for monolithic applications is far gone. Now developers are building solutions by assembling applications and services of all kinds and in all ways.

We'll show you how eZ Platform embraces the modern way of building applications by interacting with its Content Repository using
Public (PHP) API, REST API and GraphQL.
